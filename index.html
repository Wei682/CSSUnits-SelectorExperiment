3. Use .step3Class1 .step3Class2 to select all class 2 elements that are child elements of class 1 elements. Note that in this case, there is a space between the two classes, but in step 2 there's no space. No matter it is a direct child or indirect child, a background color will be added.

This is a paragraph in a parent div class named "step3Class1".

This is a paragraph in a div class named "step3Class2". The "step3Class2" div is a child of the "step3Class3" div, and the "step3Class3" div is a child of the "step3Class1" div. Though this paragraph is not a direct child of "step3Class1", it will have a background color.

This is a paragraph in a div class named "step3Class2". The "step3Class2" div is a direct child of the "step3Class1" div. It will also have a background color.

4. Use .step4Class1>.step4Class2 to select all class 2 elements that are only the direct child elements of class 1 elements.

This is a paragraph in a div class named "step4Class2". The "step4Class2" div is a direct child of the "step4Class1" div. It will have a background color.
